# Python Labs

Репозиторій з лабораторними роботами з курсу **Python** (КНУ).

## Структура

Кожна лабораторна — окрема папка:

```
Python Labs/
├── Lab1/
├── Lab2/
├── Lab3/
├── Lab4/
└── ...
```

Усередині папки лаби — файли з завданнями та, за потреби, `requirements.txt` із залежностями.

## Запуск

```bash
cd Lab4          # оберіть потрібну лабу
pip install -r requirements.txt   # якщо файл є
python3 lab4_task1.py
```
